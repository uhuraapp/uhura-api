uhura-api
=========


### Install 

With [golang](https://golang.org/) installed and configured, run:
  

    $ mkdir -p $GOPATH/src/github.com/uhuraapp 
    $ cd $GOPATH/src/github.com/uhuraapp
    $ go get
    $ go get github.com/pilu/fresh
    
### Server
    $ export PORT=3000
    $ fresh  // API is on http://localhost:3000/api
